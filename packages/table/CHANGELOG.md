## [1.5.3] - 2026-04-27

### Features

- 支持虚拟滚动列

## [1.5.2] - 2026-04-26

### Features

- 支持右键菜单功能 (#4)

### Fixed

- 列宽调整时会选中文本 (#6)

## [1.5.1] - 2026-04-18

### Fixed

- 修复客户端分页模式下，无法显示数据 (#2)

## [1.5.0] - 2026-04-18

### Fixed

- 调整属性命名
- 移除自定义滚动条样式，使用浏览器默认滚动条

## [1.4.3] - 2026-04-16

### Features

- 支持表格内容自适应宽度，新增 maxTableWidth 属性，移除 layoutMode 属性
- 代码性能优化

## [1.4.2] - 2026-04-05

### Features

- 支持汇总行功能
- 支持自定义汇总单元格

## [1.4.1] - 2026-04-01

### Breaking Changes

- 重构可编辑功能（支持行编辑和单元格编辑两种模式）
- 原 setEditingRow 改为 setEditingState，参数调整为 (rowId: string | number | null, columnKey?: string | null)
- 原 bodyCell 中的 isEditing 改为 isEditingMode，类型调整为 'row' | 'cell' | null

## [1.4.0] - 2026-03-22

### Features

- 支持行编辑功能
- 支持异步设置 tree children 属性

## [1.3.4] - 2026-03-20

### Fixed

- 展开收起表格出现刷新问题

## [1.3.3] - 2026-03-19

### Changed

- 列宽自适应优化&函数调用优化
- 官网支持国际化

## [1.3.2] - 2026-03-17

### Features

- vitepress 重构官网
- 统一各平台滚动条样式&样式细节调整

## [1.2.0] - 2026-03-11

## [1.1.1] - 2026-03-10

### Changed

## [1.1.0] - 2026-03-09

### Changed

- 支持自定义footer、加载没有更多区域、自定义展开按钮

- 支持动态配置表格主题
- VTbale 官网发布

## [1.0.0] - 2026-03-05

### Added

- 🎉 初始版本发布
- 虚拟滚动表格核心功能
- 支持排序、筛选、分页
- 支持树形结构、可展开
- 支持列宽调整，固定列
- 支持自定义单元格和表头
- 完整的 TypeScript 类型支持
- 基于 TanStack Table 和 TanStack Virtual
- 内置基础样式
