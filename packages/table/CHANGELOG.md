# Changelog

## [1.2.0] - 2026-03-11

### Changed

- 调整文档说明&打包优化

## [1.1.1-beta.1] - 2026-03-10

### Changed

- 支持 v-model 类型提示

## [1.1.1] - 2026-03-10

### Changed

- 调整部分类型定义为 interface 类型，方便做类型合并

## [1.1.0] - 2026-03-09

### Changed

- 支持自定义footer、加载没有更多区域、自定义展开按钮
- 支持表头分组及行列单元格合并
- 支持动态配置表格主题
- VTbale 官网发布

## [1.0.0] - 2026-03-05

### Added

- 🎉 初始版本发布
- 虚拟滚动表格核心功能
- 支持排序、筛选、分页
- 支持树形结构、可展开
- 支持列宽调整，固定列
- 支持自定义单元格和表头
- 完整的 TypeScript 类型支持
- 基于 TanStack Table 和 TanStack Virtual
- 内置基础样式
